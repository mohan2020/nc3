Why start from scratch?

What if we could share what we've learnt?

Showing the ROI Implementing a Nudge is more than just an idea. It requires buy-in from multiple stakeholders (executives, division chiefs, physicians, MAs, IT resources, ...). It will need also need a value proposition and more. What if there were templates you could use as a starting point?

Behavioral Economic options Opt-in? Opt-out? Peer messaging? Incentives? BE can be challenging. What if we could learn not only from each other but also the leaders in the field?

Inclusion / Exclusion criteria Nudges have to targeted to be effective. Understanding these criteria and coding it can take time (It took the Nudge Unit at Penn almmost a year in some cases). What if these learnings and the actual code developed was available to you as a starting point?

Data Extraction and Analysis Starting from a blank screen can be daunting. What if, for nudges already researched, you had a working model? Now you can tweak that to fit your patients and physicians and your ask to IT and others is based on what has already worked.

The Platform

Codifying Shared Knowledge

Library A shared repository of all Nudges - Implemented, Being Researched and Requested/Proposed. For Implemented projects, all aspects of the Nudge are available to download and re-use for members

Guides & Code Guides are available for every aspect of the implementation - pitch, value proposition, physician and other provider ask, IT ask etc. All data extratcion and analysis code will also be available.

Open to member researchers All member institutions have access to all Nudges (in various states) including currently submitted code and data. The honor system prevails i.e. please do the right thing.

Benchmarking Members are required to submit benchmarking data. The Collaborative can do anonymization and more for you or you can do it yourself.

Contributions welcome Each organization has its own EHR. We welcome, encourage and will actively support any contributions. All code contributions are shared with the community.
